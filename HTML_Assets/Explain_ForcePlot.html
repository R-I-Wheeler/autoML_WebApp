Explainability - SHAP Force Plot

The following uses SHAP (SHapley Additive exPlanations), a game theoretic approach to explain the output of any machine learning model.

...................
The dataset below shows the unseen evaluation data (10% of the modelling data), the additional columns include the ML models predictions in the column headed "label", and the probability score of the prediction in the column headed "score".
...................
A SHAP Force Plot shows how each of the features contributes to push the model output from the base value to the predicted output of the selected instance of the unseen data.
The base value is the average model output over the entire training dataset.
Features pushing the prediction higher are shown in red, those pushing the prediction lower are in blue.

The explainer plot below, shows force plots for every instance in the unseen data within a single visualisation (Up to a maximum of the first 50 rows).
By using the dropdown menus in each axis the user can view different aspects, for example the effect on the output of the model by a specific attribute on all of the predictions, or the effect one attribute has on another.
